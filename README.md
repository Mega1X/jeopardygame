# Jeopardy Game

A web-based Jeopardy game that you can run in your browser. Supports 3-team scoring and custom question loading.

**[Play the Game](https://mega1x.github.io/jeopardygame/)**

## Features
*   **3-Team Scoreboard**: Track points for up to 3 teams.
*   **Custom Questions**: Load your own questions via a JSON file.
*   **Interactive Modal**: Reveal answers and score with confirmation alerts.
*   **Responsive**: Works on desktop and mobile.

## How to Customize Questions
1.  Create a JSON file (e.g., `my_game.json`) copying the structure of `default_questions.json`.
2.  Open the game.
3.  Click **Load New Questions** and select your file.
